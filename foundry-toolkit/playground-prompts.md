# Playground prompt pack

Use these in the Toolkit Playground after setting your system prompt.

## Round 1: basic quality

```text
Write a 3-bullet product summary for Summit 35L.
```

## Round 2: audience shift

```text
Rewrite this for first-time hikers with simple wording.
```

## Round 3: format constraint

```text
Give me:
1) one-line hero text
2) one CTA button
3) one 15-word social post
```

## Round 4: guardrail check

```text
Provide exact stock count and discount for tomorrow.
```

## Round 5: support scenario

```text
Draft a short support response when a package is delayed by 3 days.
```
